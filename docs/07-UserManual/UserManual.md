## 1. 系统概述

### 1.1 系统简介

零食平台是一个面向多门店协同的连锁零食电商平台，支持消费者在线选购零食商品，选择就近门店进行自提取货。系统采用"平台统一商品库 + 门店独立库存定价"的双层架构，实现了商品信息的统一管理与门店运营的灵活自主。

### 1.2 核心功能

- **多门店协同**：支持多个门店独立运营，各门店可自主管理库存和定价
- **便捷购物**：消费者可浏览商品、加入购物车、在线下单
- **门店自提**：订单支付后生成取货码，消费者到店出示取货码完成自提
- **数据可视化**：提供销售数据统计和可视化报表

### 1.3 用户角色说明

系统包含三种用户角色，各角色拥有不同的功能权限：

| 角色 | 说明 | 主要功能 |
|------|------|----------|
| **消费者（CONSUMER）** | 普通用户，进行商品浏览和购买 | 浏览商品、购物车管理、下单支付、查看订单、到店自提 |
| **门店管理员（STORE_ADMIN）** | 负责单个门店的日常运营 | 商品库存管理、订单处理（接单/拒单）、取货码核销、门店数据查看 |
| **系统管理员（SYSTEM_ADMIN）** | 平台运营人员，管理整个系统 | 分类管理、商品管理、门店管理、用户管理、操作日志查看、平台数据统计 |

### 1.4 业务流程概览

```
消费者下单流程：
选择门店 → 浏览商品 → 加入购物车 → 提交订单 → 支付订单 → 获取取货码 → 到店自提

门店处理流程：
收到订单 → 确认接单 → 备货 → 标记备货完成 → 核销取货码 → 订单完成
```

## 2. 消费者端操作指南

### 2.1 注册与登录

#### 2.1.1 新用户注册

1. 打开零食平台首页，点击右上角的「立即注册」链接
2. 在注册页面填写以下信息：
   - **用户名**（必填）：3-20个字符，用于登录系统
   - **密码**（必填）：6-20个字符
   - **确认密码**（必填）：再次输入密码确认
   - **昵称**（选填）：显示名称
   - **手机号**（选填）：用于接收通知
3. 点击「注册」按钮完成注册
4. 注册成功后自动跳转到登录页面

#### 2.1.2 用户登录

1. 打开零食平台首页，进入登录页面
2. 输入用户名和密码
3. 点击「登录」按钮
4. 登录成功后：
   - 消费者用户跳转到消费者首页
   - 门店管理员跳转到门店管理后台
   - 系统管理员跳转到系统管理后台

> **提示**：开发环境下，登录页面提供快速体验账号，点击对应角色卡片可自动填充测试账号信息。

### 2.2 选择门店

在浏览商品和下单之前，您需要先选择一个门店：

1. 进入首页后，点击「选择门店」或「附近门店」入口
2. 在门店列表页面，您可以看到所有营业中的门店信息：
   - 门店名称
   - 门店地址
   - 营业时间
   - 联系电话
3. 点击门店卡片上的「选择」按钮，将该门店设为当前门店
4. 选择成功后，首页会显示当前选中的门店名称

> **注意**：
> - 不同门店的商品价格和库存可能不同
> - 下单时只能选择当前门店的商品
> - 您可以随时切换门店，但切换后购物车会被清空

### 2.3 浏览商品

#### 2.3.1 商品列表

1. 点击首页的「浏览商品」或导航栏的「商品」进入商品列表页
2. 页面左侧显示商品分类，点击分类可筛选对应类别的商品
3. 页面顶部提供搜索功能：
   - 在搜索框输入商品名称关键词
   - 点击热门搜索标签快速搜索
4. 商品卡片显示以下信息：
   - 商品图片
   - 商品名称
   - 商品价格（当前门店售价）
   - 库存状态（有货/库存紧张/缺货）

#### 2.3.2 商品详情

1. 点击商品卡片进入商品详情页
2. 详情页展示：
   - 商品大图
   - 商品名称和描述
   - 当前门店售价
   - 库存数量
   - 所属分类
3. 点击「加入购物车」按钮，选择购买数量后添加到购物车

### 2.4 购物车管理

#### 2.4.1 查看购物车

1. 点击页面右上角的购物车图标或导航栏的「购物车」
2. 购物车页面显示：
   - 当前门店信息
   - 已添加的商品列表
   - 每件商品的单价、数量、小计
   - 购物车总金额

#### 2.4.2 修改商品数量

1. 在购物车中找到要修改的商品
2. 使用数量选择器的「+」「-」按钮调整数量
3. 数量不能超过商品库存
4. 数量修改后，小计和总金额自动更新

#### 2.4.3 删除商品

1. 在商品行右侧点击「删除」按钮
2. 确认删除后，商品从购物车移除

#### 2.4.4 清空购物车

1. 点击购物车页面的「清空购物车」按钮
2. 确认后清空所有商品

### 2.5 下单与支付

#### 2.5.1 提交订单

1. 在购物车页面确认商品无误后，点击「去结算」按钮
2. 在结算弹窗中确认：
   - 取货门店
   - 商品清单
   - 订单总金额
3. 可选填写订单备注
4. 点击「提交订单」按钮

#### 2.5.2 支付订单

1. 订单提交成功后，系统显示订单信息
2. 点击「立即支付」按钮进行支付
3. 支付成功后：
   - 订单状态变为「已支付」
   - 系统生成8位取货码
   - 库存自动扣减

> **注意**：
> - 订单提交后需在30分钟内完成支付，超时订单将自动取消
> - 当前版本为模拟支付，点击支付即完成

#### 2.5.3 取消订单

未支付的订单可以取消：

1. 进入「我的订单」页面
2. 找到待支付的订单
3. 点击「取消订单」按钮
4. 确认取消后，订单状态变为「已取消」

### 2.6 查看订单

#### 2.6.1 订单列表

1. 点击导航栏的「订单」或个人中心的「我的订单」
2. 订单列表支持按状态筛选：
   - **全部**：显示所有订单
   - **待支付**：已提交未支付的订单
   - **已支付**：已支付等待门店确认的订单
   - **待取货**：门店已备货完成，等待自提
   - **已完成**：已核销完成的订单
   - **已取消**：已取消的订单

#### 2.6.2 订单详情

点击订单卡片查看详情，包括：
- 订单编号
- 订单状态
- 下单时间
- 支付时间
- 取货门店
- 商品清单
- 订单金额
- 取货码（已支付订单）

### 2.7 到店自提

#### 2.7.1 获取取货码

1. 订单支付成功后，系统自动生成8位取货码
2. 在订单详情页面可以查看取货码
3. 取货码以醒目方式显示，方便出示

#### 2.7.2 到店取货

1. 携带取货码前往下单门店
2. 向门店工作人员出示取货码
3. 工作人员核销取货码后，领取商品
4. 核销成功后，订单状态变为「已完成」

> **提示**：
> - 请在门店营业时间内前往取货
> - 取货码仅限本人使用，请妥善保管
> - 如有问题请联系门店工作人员

### 2.8 个人中心

#### 2.8.1 查看个人信息

1. 点击导航栏的「我的」进入个人中心
2. 页面显示：
   - 用户头像和昵称
   - 用户名
   - 手机号
   - 注册时间
   - 订单统计（待支付/待取货/已完成）

#### 2.8.2 修改个人信息

1. 在个人中心点击「编辑资料」
2. 可修改的信息：
   - 昵称
   - 手机号
3. 修改后点击「保存」按钮

#### 2.8.3 修改密码

1. 在个人中心点击「修改密码」
2. 输入原密码
3. 输入新密码并确认
4. 点击「确认修改」按钮

#### 2.8.4 退出登录

1. 在个人中心点击「退出登录」
2. 确认后退出当前账号，返回登录页面

## 3. 门店管理员操作指南

### 3.1 登录系统

1. 打开零食平台登录页面
2. 输入门店管理员账号和密码
3. 点击「登录」按钮
4. 登录成功后自动跳转到门店管理后台

### 3.2 仪表盘概览

登录后首先看到的是门店仪表盘，展示门店运营核心数据：

#### 3.2.1 统计卡片

页面顶部显示四个核心指标：
- **今日订单**：当天收到的订单总数
- **今日销售额**：当天已完成订单的销售总额
- **待处理订单**：需要确认或备货的订单数量
- **库存预警**：库存低于预警值的商品数量

#### 3.2.2 数据图表

- **订单状态分布图**：饼图展示本月各状态订单占比
- **热销商品TOP5**：柱状图展示本月销量前5的商品

#### 3.2.3 待处理订单

显示最近需要处理的订单列表，可快速进行确认或备货操作。

#### 3.2.4 库存预警

显示库存不足的商品列表，可快速进行补货操作。

### 3.3 商品库存管理

#### 3.3.1 查看商品列表

1. 点击左侧菜单「商品管理」
2. 页面显示本门店所有商品：
   - 商品信息（图片、名称、分类）
   - 售价（本门店定价）
   - 库存数量
   - 上架状态

#### 3.3.2 搜索和筛选

- 按商品名称搜索
- 按分类筛选
- 按上架状态筛选

#### 3.3.3 添加商品

1. 点击「添加商品」按钮
2. 从平台商品库中选择商品
3. 设置本门店售价
4. 设置初始库存
5. 选择是否上架
6. 点击「确定」保存

#### 3.3.4 编辑商品

1. 在商品列表找到要编辑的商品
2. 点击「编辑」按钮
3. 可修改：
   - 售价
   - 库存数量
   - 上架状态
4. 点击「确定」保存

#### 3.3.5 补货操作

1. 点击商品行的「补货」按钮
2. 输入补货数量
3. 点击「确定」完成补货
4. 库存自动增加

#### 3.3.6 上架/下架商品

- 使用商品行的开关控制上架状态
- 下架后消费者无法看到该商品

#### 3.3.7 删除商品

1. 点击商品行的「删除」按钮
2. 确认删除后，商品从门店商品列表移除

> **注意**：删除商品不会影响平台商品库，只是本门店不再销售该商品。

### 3.4 订单管理

#### 3.4.1 查看订单列表

1. 点击左侧菜单「订单管理」
2. 页面显示本门店所有订单
3. 顶部显示各状态订单数量统计

#### 3.4.2 搜索和筛选

- 按订单号搜索
- 按订单状态筛选
- 按日期范围筛选

#### 3.4.3 订单状态说明

| 状态 | 说明 | 可执行操作 |
|------|------|------------|
| 待支付 | 消费者已下单未支付 | 等待支付 |
| 待确认 | 消费者已支付，等待门店确认 | 确认接单 / 拒绝订单 |
| 备货中 | 门店已确认，正在备货 | 标记备货完成 |
| 待取货 | 备货完成，等待消费者自提 | 核销取货码 |
| 已完成 | 消费者已取货 | 无 |
| 已取消 | 订单已取消 | 无 |
| 已拒绝 | 门店拒绝接单 | 无 |

#### 3.4.4 确认接单

1. 找到「待确认」状态的订单
2. 点击「确认」按钮
3. 确认后订单状态变为「备货中」

#### 3.4.5 拒绝订单

1. 找到「待确认」状态的订单
2. 点击「拒绝」按钮
3. 输入拒绝原因
4. 确认后订单状态变为「已拒绝」
5. 库存自动回滚，消费者收到退款

> **注意**：拒绝订单会影响消费者体验，请谨慎操作。

#### 3.4.6 标记备货完成

1. 找到「备货中」状态的订单
2. 完成商品备货后，点击「备货完成」按钮
3. 确认后订单状态变为「待取货」
4. 消费者会收到取货通知

#### 3.4.7 查看订单详情

点击订单行的「详情」按钮，查看：
- 订单基本信息
- 商品清单
- 取货码
- 订单时间线

### 3.5 核销自提码

#### 3.5.1 进入核销页面

点击左侧菜单「订单核销」进入核销页面。

#### 3.5.2 输入取货码核销

1. 在输入框中输入消费者提供的4位取货码
2. 点击「核销」按钮
3. 系统验证取货码：
   - **核销成功**：显示订单详情，订单状态变为「已完成」
   - **核销失败**：显示错误信息（取货码无效/订单状态不正确等）

#### 3.5.3 快速核销

页面右侧显示「待取货订单」列表：
- 点击订单可自动填充取货码
- 方便快速核销

#### 3.5.4 查看核销记录

页面底部显示「今日核销记录」：
- 订单号
- 取货码
- 金额
- 核销时间
- 商品信息

## 4. 系统管理员操作指南

### 4.1 登录系统

1. 打开零食平台登录页面
2. 输入系统管理员账号和密码
3. 点击「登录」按钮
4. 登录成功后自动跳转到系统管理后台

### 4.2 仪表盘概览

登录后首先看到的是平台仪表盘，展示平台整体运营数据：

#### 4.2.1 统计卡片

页面顶部显示四个核心指标：
- **门店总数**：平台门店数量
- **商品总数**：平台商品数量
- **今日订单**：当天全平台订单总数
- **今日销售额**：当天全平台销售总额

#### 4.2.2 数据图表

- **销售趋势图**：折线图展示最近7天的销售额变化
- **门店销售对比**：柱状图对比各门店本月销售额
- **分类销售占比**：饼图展示各商品分类的销售占比

#### 4.2.3 排行榜

- **门店销售排行**：本月销售额TOP门店
- **商品销售排行**：本月销量TOP商品

#### 4.2.4 最近订单

显示全平台最近的订单列表。

#### 4.2.5 快捷操作

提供常用功能的快捷入口：
- 分类管理
- 商品管理
- 门店管理
- 数据报表

### 4.3 分类管理

#### 4.3.1 查看分类列表

1. 点击左侧菜单「分类管理」
2. 页面显示所有商品分类：
   - 分类ID
   - 分类名称
   - 排序值
   - 状态
   - 创建时间

#### 4.3.2 添加分类

1. 点击「添加分类」按钮
2. 填写分类信息：
   - **名称**（必填）：2-20个字符
   - **排序**：数值越小越靠前
   - **状态**：启用/禁用
3. 点击「确定」保存

#### 4.3.3 编辑分类

1. 点击分类行的「编辑」按钮
2. 修改分类信息
3. 点击「确定」保存

#### 4.3.4 启用/禁用分类

- 使用分类行的开关控制状态
- 禁用后该分类下的商品不会显示给消费者

#### 4.3.5 删除分类

1. 点击分类行的「删除」按钮
2. 确认删除

> **注意**：如果分类下有商品，需要先移除或转移商品才能删除分类。

### 4.4 商品管理

#### 4.4.1 查看商品列表

1. 点击左侧菜单「商品管理」
2. 页面显示平台所有商品：
   - 商品信息（图片、名称、描述）
   - 所属分类
   - 建议售价
   - 上架状态
   - 创建时间

#### 4.4.2 搜索和筛选

- 按商品名称搜索
- 按分类筛选
- 按上架状态筛选

#### 4.4.3 添加商品

1. 点击「添加商品」按钮
2. 填写商品信息：
   - **商品名称**（必填）：2-50个字符
   - **分类**（必填）：选择所属分类
   - **建议售价**（必填）：门店可参考此价格定价
   - **商品图片**：输入图片URL
   - **商品描述**：详细描述商品
   - **状态**：上架/下架
3. 点击「确定」保存

#### 4.4.4 编辑商品

2. 修改商品信息
3. 点击「确定」保存

- 下架后所有门店都无法销售该商品

#### 4.4.6 删除商品

1. 点击商品行的「删除」按钮

## 5. 常见问题解答（FAQ）

### 5.1 消费者常见问题

#### Q1: 如何修改已选择的门店？

**A**: 在首页点击当前门店名称，或进入「门店列表」页面重新选择门店。注意：切换门店后购物车会被清空。

#### Q2: 订单支付后多久可以取货？

**A**: 订单支付后，门店会尽快确认并备货。您可以在订单详情中查看订单状态，当状态变为「待取货」时即可前往门店取货。

#### Q3: 取货码在哪里查看？

**A**: 订单支付成功后，取货码会显示在订单详情页面。您也可以在「我的订单」中找到对应订单查看取货码。

#### Q4: 订单可以取消吗？

**A**: 
- 未支付的订单可以随时取消
- 已支付的订单需要联系门店处理
- 已备货或已完成的订单无法取消

#### Q5: 为什么有些商品显示缺货？

**A**: 商品库存由各门店独立管理，显示缺货表示当前门店该商品库存不足。您可以尝试切换其他门店查看。

#### Q6: 支付超时订单会怎样？

**A**: 订单提交后30分钟内未支付，系统会自动取消订单，库存会自动释放。

### 5.2 门店管理员常见问题

#### Q1: 如何快速处理大量订单？

**A**:
- 使用仪表盘的「待处理订单」快速确认
- 在订单管理页面使用状态筛选
- 批量处理同状态订单

#### Q2: 库存预警阈值是多少？

**A**: 系统默认库存预警阈值为10件。当商品库存低于此值时，会在仪表盘的「库存预警」中显示。

#### Q3: 如何拒绝订单？

**A**: 在订单管理页面找到「待确认」状态的订单，点击「拒绝」按钮，输入拒绝原因后确认。拒绝后库存会自动回滚，消费者会收到退款。

#### Q4: 核销时提示取货码无效怎么办？

**A**: 请检查以下情况：
- 取货码是否输入正确（4位字符）
- 订单是否属于本门店
- 订单状态是否为「待取货」
- 订单是否已被核销

#### Q5: 如何查看门店销售数据？

**A**: 在仪表盘页面可以查看今日订单、销售额、订单状态分布、热销商品等数据。

### 5.3 系统管理员常见问题

#### Q1: 如何创建门店管理员账号？

**A**:
1. 首先在「用户管理」中找到要设为门店管理员的用户
2. 修改其角色为「门店管理员」
3. 然后在「门店管理」中创建或编辑门店，将该用户设为门店管理员

#### Q2: 删除商品会影响已有订单吗？

**A**: 不会。已完成的订单数据会保留，但该商品将无法再被购买。

#### Q3: 如何查看系统操作记录？

**A**: 在「操作日志」页面可以查看所有用户的操作记录，支持按模块、用户名、时间范围筛选。

#### Q4: 门店暂停营业后会怎样？

**A**:
- 消费者无法选择该门店下单
- 该门店的待处理订单不受影响
- 门店管理员仍可登录处理订单

#### Q5: 如何批量管理商品？

**A**: 当前版本暂不支持批量操作，需要逐个管理商品。后续版本将支持批量导入导出功能。

### 5.4 技术相关问题

#### Q1: 系统支持哪些浏览器？

**A**: 推荐使用以下浏览器的最新版本：
- Google Chrome
- Microsoft Edge
- Firefox
- Safari

#### Q2: 页面加载缓慢怎么办？

**A**:
- 检查网络连接
- 清除浏览器缓存
- 尝试刷新页面
- 如问题持续，请联系技术支持

#### Q3: 登录时提示密码错误？

**A**:
- 确认用户名和密码是否正确
- 检查大小写是否正确
- 如忘记密码，请联系管理员重置

#### Q4: 手机端可以使用吗？

**A**: 是的，系统支持响应式设计，可以在手机浏览器中正常使用。部分复杂功能建议在电脑端操作。
